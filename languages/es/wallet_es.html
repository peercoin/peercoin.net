Instalación


Windows

Windows

  • Abrir o ExtraerPeercoin_v0.7.1_win.zip
  • Instalar la versión de 32 bits o 64 bits
  • El directorio de instalación predeterminado es C:\Program Files\PPCoinoC:\Program Files\Peercoin
  • Correr peercoin-qto peercoinDesde el menú Inicio o desde el directorio de instalación.

Daemon

  • Crearppcoin.confen%APPDATA%\PPCoin
  • Correrdaemon\peercoind
  • Su billetera se encuentra en C:\Users\<username>\AppData(carpeta oculta).
  • Abra la carpeta conMenú de inicio -> Ejecutar ...(o presioneWinkey-R),escriba%APPDATA%\PPCoinen el campo, y presione Enter.

Ayuda adicional se puede encontrar de peercoin en suDocumentacióno preguntando por nuestroforo
OS X

OS X

  • Abrir o Extraerpeercoin-v0.7.1ppc-osx-gitian.zip
  • MontarPeercoin-Qt.dmgabriéndolo
  • ArrastrarPeercoin-Qt.app o Peercoin.app a Aplicaciones
  • ArrastrarPeercoin-Qt.app o Peercoin.app a Aplicaciones
  • Arranque enAplicacionesy abra Peercoin-Qt.app o Peercoin.app

Daemon

  • Abrir zippeercoind.zip
  • Crear ppcoin.conf en ~/Library/Application Support/PPCoin
  • Correr peercoind
Su billetera se encuentra en ~/Library/Application Support/PPCoin. Para abrir la carpeta, presione Comando-Shift-G en el Finder, inserte la ruta y presione Enter

Ayuda adicional se puede encontrar en el PeercoinDocumentacióno preguntando por nuestroforos
Linux

Generic Linux

  • Extraer peercoin-v0.7.1ppc-linux-gitian.zip
  • correr bin/32/peercoin-qt (requires libqt4-gui)

Daemon

  • Crear peercoin.conf en ~/.peercoin
  • Correr bin/32/peercoind
Su billetera se encuentra en ~/.peercoin.
Nota: binarios de 64 bits disponibles en bin/64/{peercoin-qt,peercoind}.

ArchLinux

yaourt -S peercoin-qt

Ayuda adicional se puede encontrar en la Peercoin Documentación o preguntando por nuestroforo.
