安装


Windows

Windows

  • 打开或提取 Peercoin_v0.7.1_win.zip
  • 安装32位或64位版本
  • 默认安装目录为 C:\Program Files\PPCoin or C:\Program Files\Peercoin
  • 运行 peercoin-qt 或者 peercoin 从“开始”菜单或“安装”目录

守护进程

  • 创建 ppcoin.conf 在 %APPDATA%\PPCoin
  • 运行 daemon\peercoind
  • 你的钱包位置在 C:\Users\<username>\AppData (隐藏文件).
  • 打开文档 开始菜单 -> 运行... (或者按 Winkey-R), 输入 %APPDATA%\PPCoin 进入字段，并按Enter键。

额外的帮助可以在点点币中找到 文档 或者通过问我们 论坛。
OS X

OS X

  • 打开或提取 peercoin-v0.7.1ppc-osx-gitian.zip
  • 选择 Peercoin-Qt.dmg 并且打开它
  • 托 Peercoin-Qt.app 或 Peercoin.app 到 应用
  • 托 Peercoin-Qt.app 或 Peercoin.app 到 应用
  • 到 应用 和开放 Peercoin-Qt.app 或 Peercoin.app

守护进程

  • 解压 peercoind.zip
  • 创建 ppcoin.conf 在 ~/Library/Application Support/PPCoin
  • 运行 peercoind
你的钱包在~/Library/Application Support/PPCoin。打开文件夹，在Finder中按Command-Shift-G键，插入路径，然后回车。

额外的帮助可以在点点币中找到 文档 或者通过问我们 论坛。
Linux

通用Linux

  • 解压 peercoin-v0.7.1ppc-linux-gitian.zip
  • 运行 bin/32/peercoin-qt (需要 libqt4-gui)

守护进程

  • 创建 peercoin.conf 在 ~/.peercoin
  • 运行 bin/32/peercoind
你的钱包在~/.peercoin.
注:64位二进制文件可用 bin/64/{peercoin-qt,peercoind}.

ArchLinux

yaourt -S peercoin-qt

额外的帮助可以在Peercoin中找到 文件 或者通过问我们 论坛。
